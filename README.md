# Mana Symbols Typst Package
This is a [Typst](https://typst.app/) package for displaying [Magic the Gathering][wp:mtg] [mana symbols][mw:ms]. The package is split into two parts: the typst code (`package`) and the Rust code (`wasm`). Most of the implementation is contained in the Rust crate [`mana-symbols`][gw:ms].

[wp:mtg]: https://en.wikipedia.org/wiki/Magic:_The_Gathering
[mw:ms]:  https://mtg.wiki/page/Numbers_and_symbols#Mana_symbols
[gw:ms]: https://github.com/Pazzaz/mana-symbols
